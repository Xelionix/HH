# Harsh Habitats

![Harsh Habitats Logo](assets/png/logo.png)

Harsh Habitats is an immersive 3D experience where players navigate through challenging environments, solve puzzles, and uncover hidden secrets. Built using X3DOM and Cannon.js, Harsh Habitats offers a unique blend of stunning visuals and interactive gameplay.

## Introduction

Harsh Habitats takes players on a thrilling journey through diverse landscapes filled with obstacles and challenges. With its cutting-edge 3D rendering powered by X3DOM and realistic physics simulation using Cannon.js, Harsh Habitats delivers an immersive gaming experience like no other.

## Features

- **Immersive 3D Environments:** Explore beautifully crafted 3D environments with stunning visuals and dynamic lighting.
- **Realistic Physics Simulation:** Interact with objects and navigate through obstacles with realistic physics simulation powered by Cannon.js.
- **Challenging Puzzles:** Test your problem-solving skills with a variety of challenging puzzles and obstacles scattered throughout the game world.
- **Engaging Gameplay:** Dive into an engaging storyline and uncover hidden secrets as you progress through the game.

## Installation

To run Harsh Habitats locally, follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your web browser.

## Usage

- Use the arrow keys to navigate through the game world.
- Interact with objects by clicking or pressing the designated keys.
- Solve puzzles and overcome obstacles to progress through the game.
